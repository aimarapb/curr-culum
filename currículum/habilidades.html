Habilidades Técnicas

A lo largo de mi formación en ASIR y mediante proyectos personales, he adquirido una sólida base en diversas áreas técnicas que abarcan administración de sistemas, bases de datos y programación.

Sistemas Operativos

Linux (Ubuntu)

Experiencia en la administración de sistemas Linux, incluyendo la gestión de usuarios y permisos, instalación y configuración de software, y monitorización del sistema. Conocimientos en automatización de tareas mediante Bash scripting y administración de servicios como Apache.

Windows Server

Familiarizada con la configuración de roles y características de Windows Server, así como en la creación y gestión de Active Directory y políticas de grupo.

Bases de Datos

Oracle Database

Experiencia en la instalación y gestión de bases de datos Oracle, creación de tablas, optimización de consultas SQL y manejo de transacciones. También manejo de PL/SQL para la creación de procedimientos almacenados y funciones.

MySQL

Competencia en la creación y administración de bases de datos, ejecución de consultas complejas, y optimización del rendimiento de la base de datos

Lenguajes de Programación

JavaScript

Manejo de JavaScript para la creación de funcionalidades interactivas en páginas web. Tengo conocimientos sólidos sobre las estructuras básicas del lenguaje, la manipulación de datos y la validación de formularios. Utilizo JavaScript principalmente para añadir lógica y mejorar la experiencia de usuario en sitios web.

PL/SQL (Oracle)

Experiencia en la gestión de bases de datos Oracle utilizando PL/SQL para la creación, modificación y administración de tablas, consultas y relaciones. Puedo gestionar transacciones y asegurar la integridad de los datos, aunque mi trabajo ha estado más enfocado en el uso del lenguaje para tareas de administración que en la creación de scripts complejos.

Herramientas de Virtualización

VirtualBox

Experiencia en la creación y gestión de entornos virtuales para pruebas de sistemas operativos y aplicaciones en entornos controlados.

Administración de Redes

Configuración de Redes

Gestión de redes locales, incluyendo configuración de interfaces de red, administración de firewalls (iptables), y configuración de servicios de red como DHCP y DNS.

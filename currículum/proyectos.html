Proyectos Realizados

A continuación, destaco algunos de los proyectos más relevantes en los que he trabajado, que reflejan mis habilidades en desarrollo web, administración de servidores y seguridad informática.

Desarrollo de Páginas Web con JavaScript

Desarrollé varias páginas web dinámicas utilizando HTML, CSS y JavaScript. Las páginas incluyen formularios interactivos, validación de datos y elementos visuales animados. Estos proyectos me permitieron profundizar en el uso de JavaScript para manipular el DOM y mejorar la experiencia del usuario.

    Funcionalidades implementadas:
  • Validación de formularios y manejo de eventos.
  • Creación de menús interactivos y control de deslizamientos de imágenes.
  • Uso de JavaScript para agregar interactividad y lógica en tiempo real.

Tecnologías Utilizadas: HTML5, CSS3, JavaScript.

Resultados: Mejoras en la interactividad de las páginas y en la experiencia del usuario, logrando una navegación más fluida y atractiva.

Monitoreo y Análisis de Seguridad con Mimikatz

Realicé pruebas de seguridad y monitoreo en un entorno controlado utilizando Mimikatz, una herramienta de post-explotación para la extracción de credenciales. El objetivo del proyecto fue el análisis del tráfico de la red y entender cómo mejorar la seguridad de las contraseñas y las políticas de acceso en una red corporativa.

    Actividades Realizadas:
  • Implementación de Mimikatz para analizar y comprender el tráfico de la red
  • Análisis del impacto y recomendaciones para fortalecer la seguridad de los sistemas.
  • Captación de credenciales de usuario en una página http

Tecnologías Utilizadas: Mimikatz.

Resultados: Detección de vulnerabilidades críticas en la gestión de credenciales, con recomendaciones para fortalecer la seguridad en los servidores y estaciones de trabajo.

Configuración de Servidor DHCP en Windows Server y Debian

Implementé y configuré servidores DHCP (Dynamic Host Configuration Protocol) en dos entornos distintos: uno con Windows Server y otro con Debian. El objetivo era permitir la asignación automática de direcciones IP a los dispositivos de la red, facilitando la gestión de las conexiones.

    Actividades Realizadas:
  • Configuración y administración de un servidor DHCP en Windows Server 2019, incluyendo la creación de scopes (ámbitos de direcciones IP).
  • Instalación y configuración de un servidor DHCP en Debian, gestionando la asignación dinámica de IPs y ajustando los tiempos de concesión.
  • Pruebas de conectividad y resolución de problemas de asignación de IPs en ambos entornos.

Tecnologías Utilizadas:Windows Server 2019, Debian, DHCP, Terminal.

Resultados: Implementación exitosa de servidores DHCP en ambos sistemas, mejorando la gestión de la red y la automatización de la asignación de direcciones IP, asegurando una conectividad eficiente en el entorno de red.
